Exercice 1 : Effet Rideau (Transition)

Mon analyse

Connaissances : J'ai créé un effet de dévoilement ("Rideau"). J'ai superposé un bloc noir (position: absolute) sur une image, puis j'ai utilisé transform: translateX(-100%) au survol pour le faire glisser hors du cadre.
Difficulté : Niveau débutant. La subtilité réside dans l'utilisation de overflow: hidden sur le parent pour que le rideau disparaisse proprement.
Intérêt : Cet exercice permet de bien comprendre la différence entre transition (qui lisse le mouvement sur une durée) et transform (qui modifie la géométrie).
Qualité : L'animation est fluide grâce à la fonction de timing ease-in-out.

Exercice 5 :

Mon analyse

Connaissances : J'ai créé une bannière thématique (Fondation SCP) imitant un écran de terminal sécurisé. J'ai utilisé des gradients CSS avancés (repeating-linear-gradient) pour créer les hachures d'avertissement et l'effet de lignes "CRT" (écran cathodique) sans utiliser d'images lourdes.
Difficulté : Niveau DIFFICILE. Le défi était de superposer les couches (background, contenu, overlay CRT) avec z-index et position: absolute tout en gardant le texte lisible.
Intérêt : Cet exercice montre comment créer une ambiance forte uniquement avec du code CSS et une animation @keyframes de scintillement.
Qualité : Le JavaScript ajoute une touche finale avec un effet "machine à écrire" (Typewriter) qui simule le chargement des données de l'agent Azad Oztopal en temps réel.
