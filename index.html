Présentation du site

Ce site regroupe une sélection d’exercices réalisés pendant les TPs de Développement Front. Chaque page de TP présente au moins deux exercices accompagnés d’un commentaire personnel.

Mes acquis en développement Front

Au cours de ce semestre, j'ai acquis les fondamentaux du développement web moderne :

  • HTML5 Sémantique : Structuration logique des pages et accessibilité.
  • CSS & Box Model : Maîtrise des marges, bordures et du positionnement.
  • Mise en page moderne : Utilisation de Flexbox pour les alignements et de CSS Grid pour les structures complexes.
  • Responsive Design : Adaptation des interfaces aux mobiles grâce aux Media Queries.

Améliorations futures

À l'avenir, j'aimerais approfondir l'accessibilité web (normes WCAG) pour rendre mes sites utilisables par tous, et continuer à dynamiser mes interfaces avec du JavaScript.

Dans une démarche UX (Expérience Utilisateur), je souhaite aussi travailler davantage sur les micro-interactions et les animations CSS pour rendre la navigation plus fluide et intuitive, tout en optimisant la structure de mon code.
