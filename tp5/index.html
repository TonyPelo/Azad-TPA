Exercice 2 : Interaction JavaScript au Scroll

Mon analyse

Connaissances : J'ai utilisé l'événement JavaScript window.scroll pour détecter le défilement. J'ai appris à calculer la position de l'utilisateur en pourcentage et à manipuler le style CSS dynamiquement.
Difficulté : Niveau intermédiaire. La logique mathématique pour transformer une position de scroll en une valeur de couleur demande de la réflexion.
Intérêt : Ce type d'interaction rend le site vivant. C'est une introduction aux effets de "Scrollytelling".
Qualité : L'animation est fluide grâce à l'ajout de la propriété CSS transition: linear.

Exercice 6 : Effet Parallaxe (Javascript)

Mon analyse

Connaissances : J'ai conçu une page immersive où l'image de fond défile plus lentement que le texte. J'ai utilisé des attributs personnalisés HTML data-speed. C'est cette valeur que le JavaScript récupère (via dataset.speed) pour multiplier le déplacement : plus le chiffre est bas, plus l'image bouge lentement.
Difficulté : Niveau moyen. La complexité réside dans la formule : (scrollY - offsetTop) * speed. C'est ce calcul qui synchronise le mouvement de l'image avec le scroll.
Intérêt : L'utilisation de data-speed rend le code modulaire : je peux changer la vitesse de l'effet directement dans le HTML sans toucher au JavaScript.
Qualité : Le rendu est fluide et l'effet ajoute une réelle profondeur sans perturber la lecture.
